# Contributing to ChristmasTree

Thank you for your interest in contributing to this project! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your local machine.
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b my-new-feature
    ```
4.  **Make your changes** and commit them with clear commit messages.
5.  **Push your changes** to your fork:
    ```bash
    git push origin my-new-feature
    ```
6.  **Submit a Pull Request** (PR) to the `main` branch of this repository.

## Guidelines

*   Please ensure your code follows the existing coding style.
*   Test your changes locally before submitting.
*   If you're adding a new feature, consider updating the `README.md` if necessary.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the [Issues](https://github.com/yuanlehome/ChristmasTree/issues) section.

Thank you!
